# 🎮 Advanced Tic-Tac-Toe
A modern and interactive Tic-Tac-Toe game built using **HTML, CSS, and JavaScript**. The game features multiple themes, customizable player tokens, and both **Single Player (vs Computer)** and **Two Player** modes.

## ✨ Features
- 🤖 Play against a computer opponent
- 👥 Two-player mode
- 🎨 Multiple themes (Classic, Neon, Dark)
- 🔥 Customizable player tokens and symbols
- 📱 Responsive and user-friendly interface
- 🔄 Restart round functionality
- 🏠 Easy navigation between menu and game screens

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
